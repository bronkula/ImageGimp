OPTIONS
Add Timestamp
This option, which is off by default, allows you to add a timestamp to the end of any image url. This causes the system to load a fresh resource, and not draw from the cache. This option can cause you to alway load a fresh image, but can break on sites like blogger, which restrict access to images.

You will need to reload any pages for this effect to take place.

INPUT COMMANDS
make list - Make a list of images in the DOM and show the box if it is not already showing
make
make all
make linksadd images from links
make imagesadd images from images
make htmladd images from the dom
show box - Show the box, and show an image, or image list. Make sure to preload images before trying size matching, and remember that the order of commands matters
show
show allshow all images in the list
show {foo}show only images that include the text "foo"
show (14)show the first 14 links
show(5,16)show images from 5 to 16
Neither number is necessary
(,#) will start from the beginning
(#,) will stop at the end of the list
(,) is synonymous with show all

show [large]show images that are large
There are 5 image size categories: tiny, small, medium, large, extra large

show[medium]{foo}(5)show the first 5 medium images with the word foo
hide box - Hide the box
hide
clear list - Clear the current list and/or remove the current image
clear
clear all
clear listclear only the list
clear imageclear only the image
preload - Preload all the images from the list, this will freeze the script until the operation concludes
preload all
remove failed - Remove all images that failed to load or preload from the list
dump
keep - Keep any images containing expression
keep foo
keep gif
remove - Remove any words containing expression, rem or del
rem foo
del gif
keep head
keep tail		 - Keeps a number of images from the head or tail end of the imagelist
keept 5
keeph 23
remove head
remove tail		 - Removes a number of images from the head or tail end of the imagelist, remt/remh or delt/delh
remt 5
delh 23
replace - Replace one part of the url text with something else
{regex} [string] will replace all instances of regex with string
{foo}[bar] will replace all instances of foo with bar
nodupes - Remove duplicate images from the list
nodupes
background - Change the background color of the image box
background white
background #5522BB
fusker - Make a fusker style link list. This can be done with letters, numbers, and leading zeros. Multiple fuskers can be used at once.
http://www.example.com/image[01-05].jpg
// Will produce this list of images
http://www.example.com/image01.jpg
http://www.example.com/image02.jpg
http://www.example.com/image03.jpg
http://www.example.com/image04.jpg
http://www.example.com/image05.jpg

http://www.example.com/folder[a-c]/image[1-3].jpg
// Will produce this list of images
http://www.example.com/foldera/image1.jpg http://www.example.com/foldera/image2.jpg http://www.example.com/foldera/image3.jpg http://www.example.com/folderb/image1.jpg http://www.example.com/folderb/image2.jpg http://www.example.com/folderb/image3.jpg http://www.example.com/folderc/image1.jpg http://www.example.com/folderc/image2.jpg http://www.example.com/folderc/image3.jpg
single url - Add any one url to the list
http://www.example.com/image.jpg
